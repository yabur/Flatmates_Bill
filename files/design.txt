Title : Flatmates Bill
Description: An (app) that gets an input the (amount of a bill) for a particular (period)
and the (days) that each of (flatmates) stayed in the (house) for that period and return how much each flatmate has to pay.
It also generates a (PDF report) stating (names of flatmates), the (period), how much
each of them had to pay.

Objects: (look at the nouns)
        Object type = class

        Bill: amount is an attribute
              period is an attribute

        Flatmates: name (attribute)
                   days_in_house (attribute)

                   pays(bill) (method)
        PDFReport:
            filename
            generate(flatmate1,flatmate2,bill) method
